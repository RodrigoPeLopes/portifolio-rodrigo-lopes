Conhecimentos

HTML

Conhecimentos básicos em HTML, incluindo a estrutura básica de uma página, elementos de texto, links, imagens e formulários. Além disso, entendo o uso de atributos e como criar estruturas de tabelas e listas.

CSS

Entendo o uso de CSS para estilizar o conteúdo e aplicar layout, como selecionar elementos, trabalhar com cores, fontes e espaçamentos. Também estou familiarizado com as novidades dos novos recursos do CSS como grid e flexbox.

Javascript

Possuo habilidades básicas em JavaScript, como manipulação de elementos na página, criação de funções e eventos básicos. Além disso, tenho conhecimentos em algumas bibliotecas e frameworks como Jquery e React.

Bootstrap

Conheço o framework Bootstrap, que facilita o desenvolvimento responsivo e a criação de componentes visuais de forma rápida e eficiente, facilitando a criação de layouts para dispositivos móveis e desktop.

GitHub

Tenho familiaridade com o GitHub, um sistema de controle de versão e colaboração para projetos de código aberto. Sei criar repositórios, versionar meus projetos e trabalhar em equipe para desenvolver projetos de maneira colaborativa.

Git

Tenho conhecimento básico de uso de Git para versionamento de código, incluindo criação de branches, commits e merge. Embora ainda haja muito a aprender, estou sempre em busca de oportunidades para aprimorar minhas habilidades nessa ferramenta.
